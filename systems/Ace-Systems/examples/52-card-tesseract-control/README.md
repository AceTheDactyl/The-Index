# 52-Card Tesseract Control

Quantum Resonance deck-building card game using a 52-card holographic universe with 4D tesseract coordinates.

**[View Interactive Index](./index.html)** - Browse game pages with card gallery preview

## Overview

A deck-building card game where 52 playing cards represent vertices of a 4D tesseract (hypercube). Each card has dimensional coordinates, and gameplay involves navigating through 4D space using quantum resonance mechanics.

## Card Coordinates

The 7s of each suit represent the tesseract center (coordinate 0.0). Cards radiate outward with increasing dimensional distance from Ace to King.
