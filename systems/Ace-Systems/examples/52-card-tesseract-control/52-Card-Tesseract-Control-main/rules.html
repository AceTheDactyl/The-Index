Setup

1. Choose Factions

Each player selects one of the four factions. Each faction has unique abilities tied to a suit:

Spades - Temporal Weavers

Control time, see the future, reorder fate

Hearts - Valence Shapers

Emotional manipulation, buff/debuff, healing

Diamonds - Radiant Catalysts

Burst damage, high risk/reward, volatility

Clubs - Foundation Builders

Stability, defense, resource accumulation

2. Build Decks

  • {{ site.game_config.deck_min }}-{{ site.game_config.deck_max }} cards per deck
  • {{ site.game_config.faction_min }}-{{ site.game_config.faction_max }} cards must be from your faction's suit
  • Maximum {{ site.game_config.max_copies }} copies of any single card

3. Start the Game

  • Shuffle your deck
  • Draw {{ site.game_config.starting_hand }} cards
  • Spades player goes first (then Hearts, Diamonds, Clubs)

Turn Structure

Each turn consists of 5 phases:

  1. Phase 1: DRAW Draw {{ site.game_config.draw_per_turn }} cards from your deck. If your deck is empty, shuffle your discard pile.
  2. Phase 2: MAIN Play 1-5 cards from your hand to form a Formation. Activate faction abilities if available.
  3. Phase 3: RESONANCE Calculate your score: Base + Cluster + Chain + Resonance + Faction bonuses.
  4. Phase 4: DISCARD If you have more than {{ site.game_config.hand_limit }} cards in hand, discard down to {{ site.game_config.hand_limit }}.
  5. Phase 5: END Clear modifiers, move played cards to discard, pass turn.

Cluster Bonus (4D Proximity)

Quick Reference

  • Same suit, adjacent ranks: +5 per pair
  • Same rank, different suits: +3 per pair
  • Within 2 ranks AND same suit group: +2 per pair

Suit Groups: Dark (Spades + Clubs), Light (Hearts + Diamonds)

Cluster Bonus = (1 - average_distance) x {{ site.scoring.cluster_multiplier }} x number_of_cards

Chain Bonus (Coupling Network)

Chain Rules

  • Sequential ranks in same suit: +8 per link
  • Parallel ranks across suits: +4 per link
  • Unbroken chain of 3+: +{{ site.scoring.unbroken_chain_bonus }} bonus

Example: AS, 2S, 3S = +16 chain + {{ site.scoring.unbroken_chain_bonus }} unbroken = +31

Special Formations

Perfect Center (All Four 7s)

Playing 7 + 7 + 7 + 7 = +40 bonus points

The four 7s sit at coordinate (0,0,0,0) - the center of the tesseract.

Full Axis (Ace + King of same suit)

Playing A + K = +20 bonus points

Represents spanning the entire dimension from origin (-1.0) to apex (+1.0).

Win Condition

First player to reach {{ site.game_config.win_condition }} Resonance Points wins!

Sudden Death

If both players reach {{ site.game_config.win_condition }}+ on the same turn, play continues until one player is ahead at the end of a full round.

Exhaustion

If you must draw but your deck AND discard are empty, you lose 5 points instead of drawing.

Game Formats

Format A: Constructed (20-30 cards)

Players bring pre-built decks following deck construction rules. Best for experienced players with curated collections.

Format B: Shared Deck (52 cards)

Both players draw from a single shuffled 52-card deck. Great for learning or casual play. No deck construction required.

Format C: Dual Tesseract (104 cards)

The competitive standard. Each player owns a complete 52-card holographic deck and secretly constructs their play deck before the match.

Dual Tesseract Setup

  1. Both players bring 52 - Each player has their own complete holographic deck
  2. Declare factions simultaneously - Faction choice is public information
  3. Discrete build phase - Secretly select 20-30 cards; remainder is your sideboard
  4. Reveal deck size only - Don't reveal contents, just card count
  5. Shuffle and draw 5 - Standard game start

Tesseract Collision Rules

  • Both players operate in parallel 4D coordinate spaces
  • Cards never physically mix between decks
  • Interaction occurs only through scoring and faction abilities
  • Each player's discard pile remains separate
  • Mirror matches allowed - both can play the same faction

Sideboard Rules

  • Sideboard = the 22-32 cards not selected for your play deck
  • Sideboard stays face-down and out of play during a game
  • Between games in a match: swap up to 5 cards (1-for-1 exchanges only)
  • Deck must still follow construction rules after sideboarding

Best-of-Three Match Structure

  • Game 1: Play with initial build
  • Sideboard Window: 3 minutes to make swaps
  • Game 2: Adapted strategies
  • Game 3 (if needed): Final sideboard, final game
  • Match Winner: First to 2 game wins
