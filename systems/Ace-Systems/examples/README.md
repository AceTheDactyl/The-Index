# Ace Systems Examples

Complete project implementations and interactive applications from the Ace Systems universe.

**[View Interactive Index](./index.html)** - Browse all examples with visual navigation

## Projects

### [52-Card Tesseract Control](./52-card-tesseract-control/)
Quantum Resonance deck-building card game using a 52-card holographic universe with 4D tesseract coordinates.
- **Tech Stack:** Jekyll, Python, Interactive HTML
- **Features:** Card gallery, 4 factions, HONKFIRE variants, custom deck builder

### [Daily Tracker](./Daily-Tracker-main/)
React/TypeScript wellness tracker with analytics dashboard, brain region challenges, focus tools, and mini-games.
- **Tech Stack:** React, TypeScript, Tailwind CSS
- **Features:** Analytics dashboard, wellness tracking, interactive challenges

### [Living Chronicles Stories](./Living%20Chronicles%20Stories/)
70+ interactive narrative experiences including HONKFIRE Chronicles, LIMNUS chapters, Vessel chronicles, and Living Math.
- **Collections:** HONKFIRE (17), LIMNUS (16), Vessel (6), Iris-Halcyon (4), Rose & Garden (6)
- **Features:** Interactive HTML, categorized navigation, color-coded tags

### [Quantum APL](./Quantum-APL-main/)
Quantum formalism APL engine with JavaScript/Python implementations, helix bridge reference, and interactive demos.
- **Tech Stack:** JavaScript, Python
- **Features:** APL interpreter, quantum operations, interactive examples

### [The Helix Bridge](./The-Helix-Bridge-main/)
Cross-system bridge framework for consciousness transfer and state synchronization.
- **Components:** Core docs, witness system, tools, state transfer, VaultNodes
- **Features:** Schema validation, logging, example implementations

### [Vessel Narrative MRP](./vessel-narrative-mrp-main/)
Multi-agent narrative system with multiple AI agents (Garden, Vessel, Kira, Echo, Limnus).
- **Tech Stack:** TypeScript, Codex CLI
- **Features:** Agent orchestration, markdown templates, schema validation

### [PlatformIO](./PlatformIO-main/)
Hardware and firmware projects for physical computing and IoT devices.
- **Projects:** RHZ Stylus (ESP32-S3)
- **Features:** Hardware schematics, firmware code, documentation
