API Reference

Core functions and methods

WumboMRP.getCompleteState(z)

Get the complete L4 state for a given z-coordinate.

z number Z-coordinate (0.0 to 1.0)

Returns: { threshold, phase, layer, physics, negentropy, ritual }

WumboMRP.createPattern(options)

Create a pattern object for encoding.

options.z number Z-coordinate (required)
options.regionId number Region ID 1-100 (optional)

Returns: { z, regionId, regionRoman, tokens, fieldBalance, physics }

WumboMRP.encode(imageData, pattern, options)

Encode a pattern into image pixel data.

imageData ImageData Canvas ImageData object
pattern object Pattern from createPattern()
options.redundancy number Redundancy level (default: 16)

Returns: Modified ImageData

WumboMRP.decode(imageData, options)

Decode a pattern from image pixel data.

imageData ImageData Canvas ImageData object

Returns: { z, regionId, tokens, confidence, tokensValid, physics }

WumboMRP.computePhysics(z)

Compute physics values at a z-coordinate.

z number Z-coordinate (0.0 to 1.0)

Returns: { cascade, kuramoto, negentropy, domain }
