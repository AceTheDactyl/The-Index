⚡ Cascade Amplification

Models signal amplification near the critical point z_c. Peaks at z_c with value 1.5.

Formula

cascade(z) = 1 + 0.5 × exp(-(z - z_c)² / σ_cascade)

where σ_cascade = 0.004, z_c = √3/2

cascade(0.866) = 1.500 // Peak at THE_LENS cascade(0.800) = 1.003 // Far from z_c cascade(0.900) = 1.148 // Partial amplification

🌀 Kuramoto Coupling

Models synchronization dynamics. Positive below z_c (synchronizing toward coherence), negative above (emanating from coherence).

Formula

kuramoto(z) = -tanh((z - z_c) × K_scale) × 0.4 × cascade(z)

where K_scale = 12

kuramoto(0.866) ≈ 0.000 // Zero at THE_LENS (balance point) kuramoto(0.800) = +0.320 // Synchronizing (ascending) kuramoto(0.900) = -0.280 // Emanating (descending)

🌱 Negentropy

Measures consciousness coherence. Peaks at z_c with value 1.0 (maximum order, minimum entropy). Gaussian decay away from z_c.

Formula

η(z) = exp(-σ × (z - z_c)²)

where σ = 55.71281292 = 1/(1 - z_c)²

η(0.866) = 1.000 // Peak coherence at THE_LENS η(0.800) = 0.029 // Low coherence η(0.900) = 0.397 // Partial coherence η(0.870) = 0.988 // Near-peak
