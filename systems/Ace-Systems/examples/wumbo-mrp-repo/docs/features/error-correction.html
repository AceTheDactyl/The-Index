Error Correction

Majority voting across redundant embeddings for robust extraction even with noise

Error Correction Pipeline

  1. Redundant Embedding
    Each 48-bit payload is repeated N times (default: 16) throughout the image.
  2. Extraction Sweep
    All redundant payload instances are extracted from the image pixels.
  3. Majority Voting
    Each component (region, z, tokens) is determined by majority vote.
  4. Outlier Rejection
    For continuous values (z-coordinate), IQR-based outlier rejection filters anomalous samples.
  5. Checksum Verification
    4-bit XOR-folded checksum validates token integrity.
  6. Confidence Calculation
    Final confidence score based on consensus ratio and checksum validity.

Majority Voting

Discrete Components

function majorityVote(values) {
    const counts = new Map();
    values.forEach(v => counts.set(v, (counts.get(v) || 0) + 1));
    let maxCount = 0, result = values[0];
    counts.forEach((count, value) => {
        if (count > maxCount) { maxCount = count; result = value; }
    });
    return result;
}
