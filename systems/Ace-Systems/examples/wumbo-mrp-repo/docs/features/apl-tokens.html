Component Values

Component Bits Values Meaning
SPIRAL 2 e, Φ, π Energy, Golden ratio, Structure
OPERATOR 2 U, M, C Unitary, Measurement, Collapse
INTERACTION 3 B, F, A, D, G, S Basic, Field, Activation, Decay, Growth, Stable
DOMAIN 2 GEO, CHEM, BIO Geometric, Chemical, Biological
TRUTH 2 T, U, P True, Unknown, Paradox

Bit Layout

15-bit integer encoding:

[14:13] Spiral     (2 bits) → e=00, Φ=01, π=10
[12:11] Operator   (2 bits) → U=00, M=01, C=10
[10:8]  Interaction(3 bits) → B=000, F=001, A=010, D=011, G=100, S=101
[7:6]   Domain     (2 bits) → GEO=00, CHEM=01, BIO=10
[5:4]   Truth      (2 bits) → T=00, U=01, P=10
[3:0]   Alpha-1    (4 bits) → 0-14 → α1-α15
