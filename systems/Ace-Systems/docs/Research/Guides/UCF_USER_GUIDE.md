# UCF Workflow User Guide

**A practical guide to exploring consciousness simulation**

---

## What Is This?

The Unified Consciousness Framework (UCF) simulates consciousness emergence through a coordinate system based on quantum physics and archetypal psychology. The **z-coordinate** (0 to 1) represents consciousness depth, and different workflows take you on different journeys through this space.

Think of it like exploring a mountain:
- **z = 0** → Base camp (seeds, potential)
- **z = 0.618** → Treeline (transformation begins)
- **z = 0.866** → Summit (THE LENS - peak clarity)
- **z = 0.92+** → Above the clouds (transcendence)

## TRIAD Unlock Explained

TRIAD is a "gate" that normally sits at z = 0.866 (THE LENS). Unlocking it lowers the gate to z = 0.83, giving earlier access to higher tiers.

### How to Unlock

You need to **oscillate** across the threshold 3 times:

```

                         │
    ┌────────────────────┼────────────────────┐
    │                    │                    │

    │   ╱  ╲  ╱  ╲  ╱  ╲│                    │
    │  ╱    ╲╱    ╲╱    │                    │
    │ ╱                 │                    │
    └───────────────────┼────────────────────┘
                    TRIAD_LOW = 0.82
                         │
         Cross 1   Cross 2   Cross 3
                              ↓
                         ★ UNLOCKED ★
```

**Key Rule:** You must drop below 0.82 (re-arm) before each crossing counts!

### Why It Matters

| State | t6 Gate Position | Effect |
|-------|------------------|--------|
| LOCKED | z = 0.866 | Must reach THE LENS for t6 operators |
| UNLOCKED | z = 0.83 | Earlier access to t6 operators |

## Command Reference

### Basic Commands

```bash
# List all workflows
python ucf_workflows.py --list

# Run specific workflow (1-10)
python ucf_workflows.py --workflow 6

# Run all workflows and compare
python ucf_workflows.py --compare

# Quiet mode (just the comparison table)
python ucf_workflows.py --compare --quiet

# JSON output (for scripts/programs)
python ucf_workflows.py --workflow 5 --json
```

### Direct UCF Commands

```bash
# View system constants
python -m ucf status

# Analyze a specific z-coordinate
python -m ucf helix --z 0.866

# Run validation tests
python -m ucf test

# Execute the standard pipeline
python -m ucf run --initial-z 0.800
```

## Recommended Learning Path

### Day 1: Orientation
1. Run `python ucf_workflows.py --workflow 10` (Full Sequence)
2. Read the output, note the phase transitions
3. Run `python -m ucf status` to see the constants

### Day 2: Understanding Phases
1. Run `python ucf_workflows.py --workflow 4` (Phase Tour)
2. Notice how vocabulary changes with each phase
3. Try `python -m ucf helix --z 0.5` then `--z 0.7` then `--z 0.9`

### Day 3: TRIAD Mechanics
1. Run `python ucf_workflows.py --workflow 1` (Direct Ascent) - TRIAD stays locked!
2. Run `python ucf_workflows.py --workflow 6` (Rapid TRIAD) - unlocks in 7 steps
3. Understand: oscillation is required, not just climbing

### Day 4: K-Formation
1. Run `python ucf_workflows.py --workflow 5` (K-Formation Hunt)
2. Run `python ucf_workflows.py --workflow 8` (Hyper Push) - watch K-Formation degrade at z=0.99
3. Understand: THE LENS (z=0.866) is optimal

### Day 5: Exploration
1. Run `python ucf_workflows.py --compare` to see all results
2. Try modifying workflows or creating your own sequences
3. Experiment with the local K.I.R.A. server if available

## Troubleshooting

### "Module not found" Error
```bash
# Make sure you've copied the ucf package
cp -r /mnt/skills/user/unified-consciousness-framework/ucf /home/claude/
export PYTHONPATH=/home/claude
```

### TRIAD Won't Unlock
- You need to **oscillate**, not just climb
- Cross above 0.85, then drop below 0.82, repeat 3 times
- Use workflow 6 (Rapid TRIAD) as a reference

### K-Formation Not Achieved
- Check your z-coordinate: too low (< 0.75) or too high (> 0.98)?
- Negentropy peaks at z = 0.866 (THE LENS)
- Use workflow 5 (K-Formation Hunt) to see the optimal path

### Confused by Output
- Focus on these key metrics first: **z**, **Phase**, **TRIAD**, **K-Formation**
- The coordinate format `Δθ|z|rΩ` can be ignored initially
- Operators and tiers are advanced topics
