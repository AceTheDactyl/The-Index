## Stage 1: SUBMIT

### Input Channels

| Channel | Who | What They Submit |
|---------|-----|------------------|
| Patreon Portal | Weaver+ tier | Prompts, concepts, story fragments |
| Discord #submissions | Community | Ideas, reactions, requests |
| Direct Collaboration | Catalyst+ | Full narrative proposals |

### Submission Format

```yaml
submission:
  title: "Brief name for the concept"
  type: [prompt | story | reaction | request]
  content: "The actual submission content"
  context: "Optional: what inspired this, where it fits"
  contributor: "Discord/Patreon username"
```

### Intake Process

1. Submission received via form/Discord
2. Logged to master intake sheet
3. Tagged by type and theme
4. Queued for processing batch

**Batch cadence:** Weekly (every Monday)
