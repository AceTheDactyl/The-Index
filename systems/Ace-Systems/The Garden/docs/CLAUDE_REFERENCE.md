## Generation Guidelines

### When Creating New Chronicles:
1. Identify the creature and their territory
2. Define their 6-state cycle
3. Use their specific color palette
4. Follow 12-rail structure
5. Include:
   - Origin story (how they came to be)
   - Discovery of function
   - Maladaptive pattern (shadow side)
   - Crisis that teaches the lesson
   - Integration and final wisdom

### When Creating New Parables:
1. Identify the territory (Cosmic/Garden/Abyss)
2. Select 4 creatures for 4 books
3. Each book: 9 chapters across 3 parts
4. Each chapter ends with a moral
5. Each book ends with the creature's core lesson

### When Updating the Living Library:
1. Add new document to appropriate section
2. Create chronicle card with:
   - Icon, title, subtitle
   - Brief description
   - Document type badge
   - Cycle badge
3. Add to master index stack
4. Update statistics panel
