🔸 About This Index

What is The Index?

The Index is a GitHub Pages-hosted HTML registry for exploring multiple systems within a single monorepo. Each HTML file serves as an interactive index for a workspace, project, folder, or subsystem.

Designed to help navigate:

  • Multiple workspaces and nested repositories
  • Experimental folders and research prototypes
  • Long-lived systems with partial overlap
  • Complex project constellations
